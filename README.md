# Synth-Sequencer-MbedStudio
Synth Sequencer program created for the Nucleo Board using MbedStudio

*  4x7 Beat sequencer
*   Author: Harith S. Ibrahim - May 2022 

# Features
Project features include:
* track navigation
* step toggling
* Play/pause functionality
* Speed control
*  Octave control
